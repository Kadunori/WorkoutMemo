# 💪 5分割 筋トレ記録アプリ

スマートフォン向けの筋トレ記録 PWA（Progressive Web App）。ビルド不要の単一 HTML ファイルで動作します。

## 特徴

- **5分割メニュー** — 胸 / 背中 / 肩 / 脚 / 腕、曜日に合わせて自動選択
- **セット進行フロー** — Start → Next SET → Done の直感的な操作
- **前回重量の自動入力** — セット開始時に前回の重量を自動でセット
- **前回比の表示** — 完了後に各セットの重量・回数をチップ表示
- **休憩タイマー** — ±15s / ±30s 調整、プリセット 60 / 90 / 120 / 180 秒、アラート音＋点滅
- **メニュー編集** — 部位 × 加重方法でカテゴリを絞って種目を追加・並べ替え
- **体重記録** — 折れ線グラフで推移を確認
- **トレ履歴** — 日付別 / 部位別で過去のセッションを確認
- **CSV エクスポート** — 全記録を CSV として書き出し
- **オフライン対応** — localStorage に保存、PWA としてホーム画面に追加可能

## 使い方

1. `index.html` をブラウザで開く（またはホーム画面に追加）
2. 部位タブを選択（今日の曜日の部位を自動ハイライト）
3. 種目の **▶ Start** をタップ
4. 重量・回数を確認／修正して **Next SET →** を繰り返す
5. 最終セットで **Done ✓** をタップ → 自動保存

## デフォルトメニュー（部位 × 加重方法バランス）

各部位で **BB / DB / スミス / マシン / ISO / 自重** をバランスよく配置しています。

| 部位 | 主な種目 |
|------|---------|
| **胸** | バーベルベンチプレス・インクラインDB・スミスデクライン・ペックデック・ケーブルクロス + 三頭 |
| **背中** | 懸垂・バーベルロウ・スミスロウ・ワンハンドDB・ラットプルダウン・マシンロウ + 二頭 |
| **肩** | OHP・DBショルダープレス・スミスプレス・サイドレイズ×2・リアデルト・フェイスプル・シュラッグ |
| **脚** | バーベルスクワット・スミスハック・レッグプレス・レッグエクステンション・DB RDL・レッグカール・ヒップスラスト・カーフ |
| **腕** | 二頭（BB / DB / ISO / ケーブル） + 三頭（BB / DB / マシン / ISO）|

## 技術スタック

- HTML / CSS / Vanilla JavaScript（ライブラリ・フレームワーク不要）
- データ保存：`localStorage`
- 音声：Web Audio API（休憩タイマーのアラート音）
- PWA：`apple-mobile-web-app-capable` / `theme-color` 対応


## データ管理

- 記録・体重・カスタムメニューはブラウザの `localStorage` に保存されます
- **CSV 書き出し**：概要タブ → 「記録を CSV で書き出す」
- **全データ削除**：概要タブ → 「全データを削除」（2 回タップで確認）
- プライベートブラウジングモードでは保存されません
